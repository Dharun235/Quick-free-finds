
# Cognitive Warfare & Information Defense Resources

Curated collection of high-value, open-access resources to understand cognitive warfare, information operations, misinformation, propaganda, and defensive strategies.

---

## 1. GitHub / Open-Source Repositories

- OSINT Framework – https://github.com/lockfale/osint-framework (tools for open-source intelligence)  
- Maltego Community Edition – https://github.com/paterva/maltego (open-source intelligence mapping)  
- MISP Threat Intelligence Platform – https://github.com/MISP/MISP (information sharing & threat intelligence)  
- Social Network Analysis Tools – https://github.com/networkx/networkx (network analysis for information spread)  
- Media Bias / Fact-Checking Datasets – https://github.com/fivethirtyeight/data (including misinformation-related datasets)
